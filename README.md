## âš™ï¸ Extension Settings

Configure in **Settings** (`Ctrl+,`) or in `settings.json`:

| Setting | Type | Default | Description |
|---|---|---|---|
| `multiFolderWorkspace.defaultFolderCount` | `number` | `2` | Number of folders to select (min: 2, max: 10) |
| `multiFolderWorkspace.autoSaveWorkspace` | `boolean` | `false` | Skip the save-prompt and always save a `.code-workspace` file |

**Example `settings.json`:**
```json
{
  "multiFolderWorkspace.defaultFolderCount": 3,
  "multiFolderWorkspace.autoSaveWorkspace": true
}
```
